# 📌 Project Overview

Businesses generate thousands of sales transactions every day. Raw transactional data is difficult to understand and cannot directly support business decisions.

The objective of this project was to transform raw sales data into an interactive Business Intelligence dashboard using Microsoft Power BI. The dashboard enables stakeholders to monitor business performance, identify sales trends, compare regional performance, evaluate customer segments, and analyze product categories.

The project demonstrates data cleaning, data visualization, dashboard design, business analytics, and storytelling using Power BI.

---

# 🎯 Business Problem

Suppose you are the management team of a retail superstore.

Questions you may ask include:

- How much revenue has the company generated?
- Which states generate the highest sales?
- Which customer segment contributes the most revenue?
- Which product categories perform best?
- How has sales changed over time?
- Which quarter performs best every year?
- Which regions need improvement?

Without a dashboard, answering these questions would require manually analyzing thousands of rows of sales data.

This dashboard solves that problem by providing real-time interactive visualizations.

# ❓ Interview Questions & Answers

## Q1. Walk me through your project.

Answer:

This project is an interactive Power BI dashboard built using a U.S. Superstore sales dataset. The objective was to transform raw transactional data into meaningful business insights. I first cleaned the dataset using Power Query, verified data types, handled missing values, and then created an interactive dashboard with KPIs, maps, charts, and slicers. The dashboard allows users to analyze sales by year, state, customer segment, category, and sub-category. It helps management monitor performance and make data-driven decisions.
